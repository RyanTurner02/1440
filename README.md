# 1440

Everyone has 1440 minutes a day.

This web page is for displaying the remaining minutes of the day starting from 1440.

## Installation

1. Clone the repository and navigate to the root directory:

```bash
git clone "https://github.com/RyanTurner02/1440.git" && cd 1440
```

2. Install the dependencies:

```bash
npm install
```

## Usage

To run and use the web page:

```bash
npm run start
```

To toggle betwee light and dark mode, click on the sun/moon icon located at the top-right of the web page.
